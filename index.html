Objective:-

The Recipe Content Management System (CMS) is a dynamic and user-friendly platform developed using NodeJS and ExpressJS to provide a robust RESTful API backend. Used ReactJS and TailwindCSS to build the frontend. This project serves as a recipe-sharing platform where users from around the world can post and view various recipes. The system is designed to offer seamless management of recipes and user profiles, ensuring a smooth and interactive experience for all users.

TypeScript and an industry standard code structure is followed to enhance code maintainability.

Features:-

  • Users can create, edit, and delete recipes.
  • Each recipe can include ingredients, steps, images, and other relevant details.
  • Data updated in the backend is reflected on the frontend in real-time using query invalidation.
  • Users can create and manage their profiles.
  • The app also features image compression using html canvas natively in order to reduce quality and store data URL in the database.
  • The platform is designed to be fully responsive, ensuring a seamless experience across various devices.
